# ❄️ Winter MERN Project

## 📌 About Project
This is a full-stack MERN project.

It contains:
- Client (React + Vite)
- Server (Node.js + Express)
- MongoDB Database

---

## 🛠 Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS

## ⚙️ Installation Guide

### 1️⃣ Clone Repository
git clone https://github.com/YOUR_USERNAME/winter.git

### 2️⃣ Install Client
cd client
npm install

### 3️⃣ Install Server
cd server
npm install

---

## ▶️ Run Project

Client:
npm run dev

Server:
npm start
